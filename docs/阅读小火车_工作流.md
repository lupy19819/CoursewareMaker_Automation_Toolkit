# 阅读小火车专项生成与校验规则

> 与阅读小帆船共用脚本 `build_reading_config.py`，通过 `--type train` 区分。
> 生成策略：复制替换型 — 从基准配置 deepcopy 骨架，替换可变内容（题干音频、选项图片、放置区答案）。
> 意图判定、素材同名确认、任务清单、新建游戏、导入保存、回读比对、预览和发布都由主流程 `workflow/` 控制。

---

## 一、主流程挂载点

```
题目数据（question.json）
    ↓ python3 scripts/build_reading_config.py --type train --options 3|4 --input question.json
output/<输出路径>.json
    ↓ 后续导入保存、回读、预览、发布由主流程执行
```

---

## 二、生成脚本

**路径**：`scripts/build_reading_config.py`  
**基准配置目录**：`reference_configs/reading/train_configs_all/`

### 运行

```bash
cd CoursewareMaker_Automation_Toolkit

# 4选项小火车
python3 scripts/build_reading_config.py --type train --options 4 \
  --input data/train_questions.json \
  --output output/train_configs/my_train.json

# 3选项小火车
python3 scripts/build_reading_config.py --type train --options 3 \
  --input data/train_questions.json \
  --output output/train_configs/my_train.json
```

## 三、题目数据格式

```json
{
  "slots": [
    {
      "topic_audio": "https://...audio_1.mp3",
      "option_image": "https://...image_1.png",
      "drag_audio": "https://...drag.mp3"
    },
    ...
  ],
  "answers": {
    "1": "1",
    "2": "3",
    "3": "2",
    "4": "4"
  }
}
```

| 字段 | 说明 |
|------|------|
| `slots` | 按放置区顺序排列，每个槽的题干语音、选项图片、拖拽音效 |
| `answers` | key=放置区序号(字符串)，value=正确选项的 tag |

**选项 tag 规则**：
- 4 选项：`1` / `2` / `3` / `4`
- 3 选项：`a` / `b` / `c`

---

## 四、组件渲染顺序

**⚠️ 渲染层级 = `components[]` 数组顺序（越靠后越在上层），zIndex 字段仅作编辑器排序参考，不影响运行时渲染。**

阅读小火车为「复制替换型」游戏，**所有组件顺序来自基准配置 deepcopy，不在生成脚本中重排**。只要基准模板的 `components[]` 顺序正确，生成结果即正确。

标准层级顺序（从底层→顶层，即数组从前→后）：

```
[背景] → [装饰层] → [题干音频(音频播放按钮1..N)]
→ [拖拽放置区1..N] → [拖拽选项1..N(MDraggable)]
```

> 选项（MDraggable）必须在放置区（LDragPlace）之后，保证可拖拽交互。

---

## 五、生成策略

**复制替换型**（非动态布局）：

1. 从 `reference_configs/reading/train_configs_all/` 加载匹配选项数的基准配置
2. deepcopy 整个关卡骨架
3. 替换可变内容：
   - 题干音频：`音频播放按钮N` 组件「按下」state → `MAudio.value`
   - 选项图片：`MDraggable` 组件默认/拖拽中/放置 state → `MSprite.value`
   - 拖拽音效：`MDraggable` 组件拖拽中/放置 state → `MAudio.value`
   - 放置区答案：`LDragPlace.itemList` → 正确答案 tag
4. **不重算坐标** — 所有组件位置、尺寸沿用基准配置原值

## 八、调试备忘

| 问题 | 原因 | 解法 |
|------|------|------|
| 选项数量不匹配 | 基准配置选项数与 `--options` 参数不一致 | 确认基准配置路径和参数一致 |
| 放置区 itemList 为空 | 未正确设置 `answers` 映射 | 检查 `answers` key 与放置区序号对应 |
| 题干音频不播放 | `topic_audio` 为空或 URL 失效 | 确认音频 URL 可访问 |
| 选项图片不显示 | `option_image` 为空或 URL 失效 | 确认图片 URL 可访问 |
