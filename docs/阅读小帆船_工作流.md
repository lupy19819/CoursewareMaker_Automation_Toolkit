# 阅读小帆船 配置生成工作流

> 与阅读小火车共用脚本 `build_reading_config.py`，通过 `--type fanboat` 区分。
> 生成策略：复制替换型 — 从基准配置 deepcopy 骨架，替换可变内容（题干音频、选项图片、放置区答案）。

---

## 一、整体流程

```
题目数据（question.json）
    ↓ python3 scripts/build_reading_config.py --type fanboat --options 3|4 --input question.json
output/<输出路径>.json
    ↓ python3 scripts/upload_game_config.py <game_id> <config.json>
CoursewareMaker 平台保存
```

---

## 二、生成脚本

**路径**：`scripts/build_reading_config.py`  
**基准配置目录**：`output/fanboat_configs_all/`

### 运行

```bash
cd CoursewareMaker_Automation_Toolkit

# 4选项小帆船
python3 scripts/build_reading_config.py --type fanboat --options 4 \
  --input data/fanboat_questions.json \
  --output output/fanboat_configs/my_fanboat.json

# 3选项小帆船
python3 scripts/build_reading_config.py --type fanboat --options 3 \
  --input data/fanboat_questions.json \
  --output output/fanboat_configs/my_fanboat.json
```

## 三、题目数据格式

```json
{
  "slots": [
    {
      "topic_audio": "https://...audio_1.mp3",
      "option_image": "https://...image_1.png",
      "drag_audio": "https://...drag.mp3"
    },
    ...
  ],
  "answers": {
    "1": "b",
    "2": "a",
    "3": "d",
    "4": "c"
  }
}
```

| 字段 | 说明 |
|------|------|
| `slots` | 按放置区顺序排列，每个槽的题干语音、选项图片、拖拽音效 |
| `answers` | key=放置区序号(字符串)，value=正确选项的 tag |

**选项 tag 规则**：
- 4 选项：`a` / `b` / `c` / `d`
- 3 选项：`1` / `2` / `3`

---

## 四、组件渲染顺序

**⚠️ 渲染层级 = `components[]` 数组顺序（越靠后越在上层），zIndex 字段仅作编辑器排序参考，不影响运行时渲染。**

阅读小帆船为「复制替换型」游戏，**所有组件顺序来自基准配置 deepcopy，不在生成脚本中重排**。只要基准模板的 `components[]` 顺序正确，生成结果即正确。

标准层级顺序（从底层→顶层，即数组从前→后）：

```
[背景] → [装饰层] → [题干语音(语音1..N)] → [拖拽放置区1..N]
→ [拖拽选项1..N(MDraggable)]
```

> 选项（MDraggable）必须在放置区（LDragPlace）之后，保证可拖拽交互。

---

## 五、生成策略

**复制替换型**（非动态布局）：

1. 从 `output/fanboat_configs_all/` 加载匹配选项数的基准配置
2. deepcopy 整个关卡骨架
3. 替换可变内容：
   - 题干语音：`语音N` 组件的「点击按下」state → `MAudio.value`
   - 选项图片：`MDraggable` 组件的默认/拖拽中/放置 state → `MSprite.value`
   - 拖拽音效：`MDraggable` 组件拖拽中/放置 state → `MAudio.value`
   - 放置区答案：`LDragPlace.itemList` → 正确答案 tag
4. **不重算坐标** — 所有组件位置、尺寸沿用基准配置原值

## 六、基准配置

| 选项数 | 基准游戏名 | 说明 |
|--------|-----------|------|
| 3 选项 | `国际启F融阅读小帆船秋8` | 3槽标准版 |
| 4 选项 | `国际启蒙阅读小帆船秋8` | 4槽标准版 |

基准配置存放在 `output/fanboat_configs_all/`，由线上配置抓取脚本获取。

## 七、保存到平台

```bash
python3 scripts/upload_game_config.py <game_id> output/fanboat_configs/my_fanboat.json
```

## 八、调试备忘

| 问题 | 原因 | 解法 |
|------|------|------|
| 选项数量不匹配 | 基准配置选项数与 `--options` 参数不一致 | 确认基准配置路径和参数一致 |
| 放置区 itemList 为空 | 未正确设置 `answers` 映射 | 检查 `answers` key 与放置区序号对应 |
| 题干语音不播放 | `topic_audio` 为空或 URL 失效 | 确认音频 URL 可访问 |
| 选项图片不显示 | `option_image` 为空或 URL 失效 | 确认图片 URL 可访问 |
